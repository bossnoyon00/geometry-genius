DOM stands for Document Object Model. It is a programming interface that allows us to create, change, or remove elements from the document. We can also add events to these elements to make our page more dynamic.

Event bubbling directs an event to its target. It works like this: When an element (like a button) is clicked, an event is directed to the element. If an event handler is set for the element, the event handler is triggered. Then the event "bubbles up" to the elements parent. If an event handler is set for the parent, this event handler is triggered. The event continues to bubble, until it reaches the top.

If you pass another function to a function as a parameter, it is called a callback function. These callback functions are often used in libraries and frameworks, such as the JavaScript applications jQuery, Angular, and Node.js. They are designed to extend functions and be called back for specific events or circumstances.
